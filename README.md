# html-webpack

A simple webpack settings for make static sites

## Instructions

_With [azk](http://docs.azk.io/)_

```
azk start && azk logs -f
azk shell -- npm run build
```

_Without azk_

```
npm install
npm run start:dev
npm run build
```
